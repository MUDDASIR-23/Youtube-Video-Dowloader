## 🚀 Quick Start

### Local Setup
```bash
# Clone the repository
git clone https://github.com/MUDDASIR-23/Youtube-Video-Dowloader.git
cd Youtube-Video-Dowloader

# Open in browser
open index.html
# or simply double-click index.html
```

### Netlify Deployment

**Option 1: Direct GitHub Integration**
1. Go to [Netlify](https://netlify.com)
2. Click "New site from Git"
3. Connect your GitHub account
4. Select `MUDDASIR-23/Youtube-Video-Dowloader`
5. Deploy!

**Option 2: Using Netlify CLI**
```bash
npm install -g netlify-cli
netlify deploy
```

## 🐛 Troubleshooting

### Issue: Video not found
- Verify the URL is correct
- Ensure the video is public/not restricted
- Try refreshing the page

### Issue: Download not starting
- Check internet connection
- Try a different quality
- Clear browser cache

### Issue: Slow download
- Select lower quality
- Check internet speed
- Try during off-peak hours
